# 📚 고교 생활 가이드 3D 뷰어 시스템

## 목적
Turn.js를 사용한 PDF 플립북 뷰어로 자연스러운 웹·앱 UX 노출이 가능합니다.

## 파일 구조

```
School_Guide_Book_Design/
├── school-guide-book.html    # 플립북 뷰어 메인 파일
├── pages/                    # 페이지 파일들
│   ├── page-title.html       # 표지 페이지
│   ├── page-01.html          # 첫번째 페이지
│   └── page-02.html          # 두번째 페이지
├── styles.css                # 공통 스타일시트
└── README.md                 # 프로젝트 설명서
```

## 사용 방법

1. 깃허브로 배포한 웹 브라우저에 접속
- [https://rkdekdls.github.io/School_Guide_Book_Development/]
2. 좌/우 화살표 버튼 또는 키보드 ⇦ / ⇨ 로 페이지 넘기기
3. 하단 슬라이더로 원하는 페이지 이동 및 상태 확인 가능
